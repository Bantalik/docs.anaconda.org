The token name must be unique to your account ### Remove a token binstar auth --remove 'my-unique-name' ### Scopes You can limit the scope of operations that a token can perform. For example if you are using conda or pypi To list all scopes: binstar auth auth --list-scopes To create a new token run with a restricted scope: binstar auth --create --name 'my-scoped-token' --scopes 'repo conda' {% endcall %} {% call section('channel') %} ## Manage your binstar channels {% endcall %} {% call section('config') %} ## Binstar configuration {% endcall %} {% call section('copy') %} ## Copy packages from one account to another {% endcall %} {% call section('groups') %} ## Manage Groups {% endcall %} {% call section('login') %} ## Authenticate a user {% endcall %} {% call section('logout') %} ## Log out from binstar {% endcall %} {% call section('package') %} ## Package utils {% endcall %} {% call section('register') %} ## Register a package on binstar {% endcall %} {% call section('remove') %} ## Remove an object from binstar {% endcall %} {% call section('search') %} ## Search binstar {% endcall %} {% call section('show') %} ## Show information about an object {% endcall %} {% call section('upload') %} ## Upload a file to binstar {% endcall %} {% call section('whoami') %} ## Print the information of the current user {% endcall %} {% endblock %}
