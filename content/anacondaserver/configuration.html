--- title: Client Configuration sub_items: - id: binstar-cli title: Binstar Client - id: conda-installer title: Conda Installer - id: pip-installer title: pip Installer --- {% from "macros.html" import section, subsection %} {% extends "layout.html" %} {% block main %} {% call section('binstar-cli') %} ### Binstar Client Set your local conda configuration to point to your On-site Binstar server. This is need to correctly upload your conda build packages to your On-site Binstar installation. ``` binstar config --set url http://:/api ``` {% endcall%} {% call section('conda-installer') %} ### Conda Installer To install conda packages from your On-site Binstar installation add your On-site Binstar channel to your `~/.condarc` configuration file. ``` conda config --add channels http://:/conda/ ``` {% endcall%} {% call section('pip-installer') %} ### pip Installer To install pypi packages from your On-site Binstar installation add your On-site Binstar channel to your `~/.pip/pip.conf` configuration file. Edit the file and add an `extra-index-url` entry to the global config section: [global] extra-index-url = http://:/pypi//simple {% endcall%} {% endblock main %}
